Welcome to Popcorn Maker

Let's make some popcorn.

Popcorn Maker is a fun and easy way to make interactive open video pages. Here's how to get started:

  • 1. Make a new project from the welcome screen, or press the plus button (+) from within the app.

    Pick a template. Remember, you can edit the final HTML when you are done to modify the design

  • 2. Use Popcorn Maker to add actions to the timeline of your video.

  • 3. When you're done, export using the save button

For more info, checkout the user manual
